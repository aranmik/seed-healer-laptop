# Seed Healer — Demo v1

세 가지 위협을 상대로 스킬을 준비하고, 전장을 읽고, 사제로 파티를 살려내는 **3보스 Vertical Slice**.
당신은 공격하지 않는 사제 **아리아(ARIA)**. 동료들이 싸우는 동안, 당신의 손끝이 파티의 생사를 가른다.

---

## 플레이 방법 (Demo 정의)

1. **마을**에서 **길드 게시판**으로 가 세 토벌 중 하나를 고른다.
2. **성소**에서 9개의 기도(스킬) 중 **6개를 장착**한다 — 보스마다 잘 맞는 준비가 다르다.
3. **전투**: 상단 **위협 대기열**로 곧 올 위험(강타/해일/침식 등)을 읽고, 하단 스킬로 **보호막·치유·정화·서약·씨앗·은총**을 걸어 파티를 지킨다.
4. 사제가 쓰러지거나 동료가 전멸하면 패배. 보스 HP를 0으로 만들면 승리.

**핵심 손맛:** 보스가 전장에 질문을 던지고 → 당신이 전장을 읽고 → 스킬로 답하고 → 질문과 답이 부딪히고 → HP와 마나로 결과가 남는다.

## 이번 Demo에 담긴 것

- **보스 3종**
  - 🪨 대지뿌리 골렘 — 예고된 강타로 전열을 무너뜨리는 예고형
  - 🌊 물결 성소의 정령 — 잔파도·침식으로 파티 전체를 서서히 깎는 지속 압박형
  - 🐍 나가 워리어 — 집요한 처형 베기로 한 명을 끝까지 몰아붙이는 처형형
- **고정 파티 4명**
  - ✨ 사제 아리아 (당신 · 공격하지 않음)
  - 🛡️ 방패 전사 엘리 (강타를 받아내는 탱커)
  - 🗡️ 차단 도적 소른
  - 🔥 화염 마법사 루미나
- **9 스킬 풀 / 6 장착**
  빠른치유 · 보호막 · 정화 · 구원 · 지속 · 고리 · 수호의 서약 · 기도 씨앗 · **은총의 순간**
  (은총의 순간 = 마나 0·긴 재사용, 8초 안 다음 기도 1회를 무료로 — 마나 운영의 전략 스킬)
