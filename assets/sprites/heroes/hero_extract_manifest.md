## 검수 방법 / 결과
- **캔버스**: 6/6 = 1448×1086 (원본 동일, 트림 없음).
- **코너 투명**: 6/6 네 모서리 알파 0.
- **마젠타 잔여**: 6/6 = 0 (불투명 픽셀 중 키 근접 d<70 = 0).
- **손상 정밀검사(핵심)**: 코너 플러드필로 "내부 폐쇄 투명영역"을 찾고, 각 픽셀의 **원본 색**을 대조 →
  - 원본이 마젠타(d≤120) = 정상 폐쇄 배경(팔·다리·지팡이·로브 사이). eli 65 / thorne 1987 / lumina 72 / rorin 6229 / cael 3917 / aria 5809 — 전부 정상.
