### 전투 코어
- 결정론 헤드리스 전투 코어(사제 1 + 고정 파티 3 vs 보스). 마나·시전·보호막·지속·속박·토템·덫·압박 페이즈.
- 봇 baseline(botSim) + 보스 probe(probeSim)로 손맛 회귀 자동 검증.

### 보스 3종
- 대지뿌리 골렘(예고 강타형) · 물결 성소의 정령(지속 압박형) · 나가 워리어(처형형).
- 보스별 색·형태·리듬으로 분리된 telegraph/공격 FX, 보스별 용어 매핑.

### 사제 스킬 9풀 / 6장착
- 기본 6종(빠른치유·보호막·정화·구원·지속·고리) + 신규 3종(수호의 서약·기도 씨앗·은총의 순간).
- **은총의 순간**: 마나 0·긴 재사용, 8초 안 다음 성공한 기도 1회를 무료로 — 마나 운영 전략 스킬.
- 성소 tap-to-swap 장착. 스킬별 고유 시각 문법(정화/지속/서약/씨앗 식별력).

### 전투 정보 전달 / UI
- **위협 대기열**: 곧 올 위협(최대 3)이 독립 행으로 줄 서고, 각 위협이 개별로 진행·해결·퇴장.
- **전장 결과 토스트**: 방금 해결된 결과(보호막 차단·정화·씨앗 개화·무료 시전·직격)를 전장 위에 짧게 띄웠다 사라짐.
- 머리 위 행동 표기(signature 공격), 하단 command UI(큰 아이콘 + 마나 비용 pill).

### 밸런스 / FX

- signature FX(강타/처형/해일 등) 한 단계 과장·특별한 순간만.
