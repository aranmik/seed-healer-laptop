# P1-A HANDFEEL CHECKSHEET — Seed Healer

**Laptop Carry Pack 01** · 작성: 렌 · 2026-07-04
노트북 이식 후 나라님이 **직접 손으로 플레이하며** 체크. 이 항목들이 서면 "같은 게임"이 옮겨진 것.

> 판정 보정: 나라님은 상위 0.1% 레이더 → 체감보다 한 단계 낮춰 읽을 것. "살짝 쉽다" 느끼면 적정.

---

## A. 수치 목표 (봇/리포트로 확인)

- [ ] 전투 시간이 **약 135~160초**에 든다 (표준 파티 기준)
- [ ] 좋은 플레이 시 남은 마나 **10~25%** (넉넉하지 않다)
- [ ] 오버힐 **2~8%** (필요한 만큼만 힐하면 마나가 남는 인과)
- [ ] 스팸 플레이는 후반 마나 파산으로 무너진다
- [ ] 방치하면 60초 안팎에 전멸한다

## B. 강타 — 이득이되 강요 아님 (재미의 심장)

- [ ] 강타 대상이 **즉시 보인다** (🎯 + 대상 프레임 앰버 강조 + 예고 바)
- [ ] 보호막을 걸면 **대비 완료가 읽힌다** (예고 바가 녹색 "🛡 강타 대비 완료")
- [ ] 보호막을 걸면 강타 피해가 **눈에 띄게 줄어든다** (이득 성립)
- [ ] 보호막 없이 맞아도 **불합리하게 즉사하지 않는다** (강요 아님)
- [ ] 강타 놓쳐도 빠른 치유·지속 회복·구원으로 **비벼서 수습할 창**이 있다
- [ ] (숙련 목표) 강타에 보호막 제대로 걸면 **전원 생존**이 가능하다

## C. 마나·긴장

- [ ] 마나가 **남아돌지 않는다**
- [ ] 마지막 30초에 **손이 바쁘다** (소프트 압박)
- [ ] 마지막 압박은 압박이 **급상승**할 뿐 즉사가 아니다
- [ ] 깊은 호흡을 장착하면 마나가 확연히 여유로워진다 (로드아웃 선택이 의미 있음)

## D. 전사 사망 긴장

- [ ] 전사를 방치하면 **죽을 수 있다** (긴장 요소로 살아 있음)
- [ ] 그러나 벼랑에서 **붙잡아 살릴 수도** 있다
- [ ] 탱커 없이 편성하면 확연히 어렵다 ("방패가 버티면 모두가 버틴다" 성립)

## E. 읽힘 (1차 손판정 개선분 — 절대 후퇴 금지)

- [ ] **스킬 거부 사유가 즉시 이해된다** — 안 나가면 슬롯 위 빨간 배지("6s"/"마나 부족" 등)
      ※ 이게 없으면 "왜 캔슬됐지?" 오인지가 재발한다 (나라님 실기에서 실제 발생했던 문제)
- [ ] 속박이 뜨면 **⛓️ 칩이 잘 보인다** (확대·맥박)
- [ ] 정화 타이밍을 놓치지 않을 만큼 칩이 눈에 걸린다
- [ ] 레이드 프레임 4칸·수치가 **390px 폰에서 읽힌다** (팔 길이 거리)

## F. 거울 (재도전 욕구)

- [ ] 결과 화면을 보고 **다음 판을 더 잘하고 싶다**
- [ ] 기록실 원인 문장이 **내 플레이를 정확히 비춘다** (예: "82초 강타를 보호막 없이 받았습니다")
- [ ] 하이라이트 칩이 내 스타일을 반영한다 (전원 생존/위기의 손/마나 장인 등)

---

## 종합 판정

- **PASS**: A~F 대부분 체크 + B(강타 이득·강요 아님)와 E(거부 사유 읽힘)가 확실.
- **PARTIAL**: 수치는 맞는데 읽힘(E)이 무너짐 → 배지·강조 재구현 확인.
- **FAIL**: 봇 방향이 어긋남(스팸이 이기거나 방치가 안 짐) → tuning/루프 대조(P1A_06·03).
