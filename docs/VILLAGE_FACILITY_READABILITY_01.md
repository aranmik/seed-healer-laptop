# Village Facility Readability 01

작업일: 2026-07-08 · 담당: 렌(Dev) · 판정: PASS(렌 자체) · 유키PD 확인 대기

## 목적
새 시설 기능 구현이 아니라, 현재 존재하는 마을 5시설의 **역할 / placeholder 여부 / 다음 행동**이
플레이어에게 더 잘 읽히도록 문구·라벨·상태 표시만 최소 정리한다.

핵심 문장: **"마을은 다음 전투의 답을 준비하는 곳이다."**

## 시설별 역할 기준 (유키PD)
| 시설 | 역할 |
|---|---|
| 길드 게시판 | 다음 전투를 고르는 곳 |
| 여관 | 동료를 확인하는 곳 |
| 기도소 | 사제 스킬을 준비하는 곳 |
| 기록실 | 전투 결과/기록을 보는 곳 |
| 전투 준비 | 출정 전 마지막 확인 |

## 변경 내역 (index.html 문구만)
### 마을 허브 카드 (f-desc)
- 여관: `함께 갈 동료 3명` → `함께 싸울 동료 확인`
- 기도소: `가져갈 기도 6개` → `사제의 기도 확인`
- 기록실: `지난 전투의 거울` → `전투 기록 · 준비 중`  ← placeholder임을 카드에서부터 명시
- 전투 준비: `요약 확인 후 출발` → `출정 전 마지막 확인`  ← 역할 문장과 일치
- 길드 게시판: `토벌 계약을 고른다 — 지금은 대지뿌리 골렘` (유지, 이미 명확)

### 시설 화면 sub / 라벨
- 여관 sub: `함께 싸울 동료들이 쉬고 있다.` → `이번 전투를 함께 싸울 동료를 확인한다.`
- 여관 panel: `현재 파티 (3명)` → `현재 파티 (3명) · 확인용`
- 여관 wip: `편성 변경은 준비 중` → `동료 교체·편성은 준비 중`
- 기도소 sub: `전투에 가져갈 기도를 고르는 곳.` → `전투에 가져갈 사제의 기도를 확인한다.`
  - ★"고르는 곳"은 실제 없는 선택 기능을 암시 → "확인한다"로 교정
- 기도소 panel: `장착 중인 기도 (6개)` → `가져갈 기도 (6개) · 확인용`
  - ★"장착"이 장착 시스템을 암시 → "가져갈…확인용"으로 완화
- 기도소 wip: `기도 변경은 준비 중` → `기도 장착·변경은 준비 중`
- 게시판 sub: `토벌 계약이 붙어 있다.` → `다음에 나설 토벌을 고른다.`
- 전투 준비 sub: `이 답안으로 나간다.` → `출정 전 마지막 확인 — 이 답안으로 나간다.`

### 유지(이미 정직한 placeholder — 무변경)
- 게시판 lockq: `다음 계약 — 아직 붙지 않았다`
- 기록실 화면: `아직 기록이 없다 / 전투의 결과가 이후 이곳에 쌓일 예정`
- 각 시설 `마을로 돌아가기`, 게시판 `도전하기`, 전투 준비 `전투 시작` 버튼(핸들러·흐름 불변)
