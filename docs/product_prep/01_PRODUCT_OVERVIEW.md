# 01 · PRODUCT OVERVIEW — Seed Healer

**Product Prep Pack 01 (1/7)** · 기준: 유키PD 확정 Lock 01~11 + 기준 시안 v0.1 · 작성: 렌 · 2026-07-04
> 이 문서 세트는 **Seed Healer 전용**이다. 다른 프로젝트와 비교하거나 섞지 않는다.

---

## 1. 한 줄 정의

**Seed Healer는 나(사제)가 공격하지 않고, 자동으로 싸우는 동료 3명을 실시간으로 살리는 모바일 세로형 힐러 전투 게임이다.**

## 2. 핵심 문장

> **나는 공격하지 않는다.**
> **하지만 전투의 결과는 내 손에 달려 있다.**

제품 전반의 감각 기준이 되는 보조 문장:

- 죽은 뒤에 복구하는 게임이 아니라, **죽기 직전까지 붙잡는 게임.**
- 보스를 읽고, 동료를 고르고, 기도를 준비하고, 살리러 간다.
- 마을은 다음 전투의 답안을 짜는 장소다.

## 3. 플레이어 판타지

| 항목 | 내용 |
|---|---|
| 나는 누구인가 | 전지전능한 UI 조작자가 아니라 **전장 안의 사제**. 나도 HP가 있고, 전체 공격·어그로 이탈·후열 찍기에 맞을 수 있다 (Lock 02) |
| 나의 자리 | 레이드 프레임 **1번 칸이 곧 나**. 파티의 심장이자 마지막 방어선 |
| 나의 승리 | 동료들이 각자의 역할을 끝까지 하게 만든 것. 킬 로그에 내 이름은 없지만, **모두가 서 있는 것**이 내 전적 |

## 5. 주요 차별점

1. **힐러가 주인공인 실시간 전투** — 공격 버튼이 0개인데 전투를 지배한다.
2. **사제도 전장 안에 있다** — 외부 시점의 조작자가 아니라, 나도 맞고 나도 살아남아야 하는 파티원이다.
3. **결정론 전투** — 난수가 아니라 조건. 같은 판단은 같은 결과. 리트라이가 곧 학습이 된다. (R&D 두 씨앗 연속 검증)
4. **뽑기 없음** — 동료는 구하고, 설득하고, 계약해서 데려온다. 스킬은 기도소와 여정으로 열린다. (Lock 06·07)
5. **실패는 분석** — 실패 보상 파밍이 아니라, 기록실의 원인 분석이 다음 도전을 설계한다. (Lock 09·10)
6. **성장은 정확함** — Lv.99식 숫자 폭증이 아니라 MAX 3단계. "더 강한 사제가 아니라, 더 정확히 살리는 사제." (Lock 08)

## 6. 플랫폼 / 화면 방향

- **모바일 세로(portrait) 전용.** 레이아웃 기준 390×844, 한 손 엄지 조작 우선.
- 전투 길이: 일반 짧은 전투 40~50초 / 인지된 보스전 기본 약 2분 30초 / 특수 장기전·챕터 보스 최대 약 5분. **전부 손맛 튜닝 대상.** (Lock 09)
- 아트: 몽글몽글한 2~3등신 동화풍 JRPG. 따뜻한 성소의 감성, 그러나 **보스 전조는 명확히 읽힌다.** (→ 06 문서)

## 7. 핵심 루프

```
마을 (답안 작성)
 → 길드 게시판 (계약 선택 · 위험 태그 읽기)
 → 여관 (3인 편성) · 기도소 (6슬롯 장착)
 → 전투 (실시간 힐링 실행)
 → 결과 (판정 · 보상 · 하이라이트)
 → 기록실 (분석 · 도감)
 → 마을 (다음 답안)
```

- 감정 곡선: **안전(마을) → 긴장(전투) → 해소(결과) → 통찰(기록실) → 다시 안전.**
- 실패 시에도 동일 루프: 실패 → 결과 리포트 → 마을 복귀 → 재준비 → 재도전 (Lock 09).

| 02 | 사제 역할 | 공격 불가·피격 가능. 치유/보호/정화/구원/자기 생존/마나 관리 |

| 05 | 마을 허브 | 5시설(여관·기도소·게시판·기록실·시장). 감성 풍부·조작 짧게·선택 선명 |
| 06 | 동료 획득 | 스토리·보스 보상·구출·길드 계약. 가챠 중심 금지. 초반 5직업 1명씩 개방 |
| 07 | 스킬 획득 | 기도소/보스 보상/신앙 시험/챕터 해금. 스킬 가챠·희귀도·중복 조각 금지 |
| 08 | 성장 | MAX 3단계 + 이후 2특성 택1. 축: 사제 신앙·스킬 숙련·동료 인연 |
| 09 | 진행 | 보스 계약형+챕터형. 정답 전체 공개 금지 — 힌트·위험 태그·도감 해금 |

| 11 | 초반 30분 | "공격하지 않는 사제가 전투를 지배할 수 있다"는 확신을 주는 시간 |
