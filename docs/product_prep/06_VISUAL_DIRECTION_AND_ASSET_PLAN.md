# 06 · VISUAL DIRECTION & ASSET PLAN — Seed Healer

**Product Prep Pack 01 (6/7)** · 기준: 기준 시안 v0.1 + 유키 요청서 §4 아트 방향 + 루미 협업 처방전(세이브 §6c) · 작성: 렌 · 2026-07-04

## 1. 전체 아트 방향

> 몽글몽글한 2~3등신의 귀여운 JRPG풍.
> 따뜻한 동화 감성과 성소/기도의 분위기.
> **하지만 보스의 위협과 전조는 명확히 읽혀야 한다.**
> 모바일 세로 화면에서 **작은 크기로도 식별 가능**해야 한다.

시안 v0.1의 Design Pillars를 그대로 5원칙으로 승계한다.

| # | 원칙 | 제작 기준으로 번역 |
|---|---|---|
| 1 | Clarity over clutter | 화면당 정보 계층 3단 이내. 장식이 판독을 이기면 장식을 뺀다 |
| 2 | Healer fantasy | 사제가 모든 구도의 중심 (타이틀·레이드 프레임 1번 칸·결과 MVP) |
| 3 | Warm & welcoming | 마을·UI는 크림/골드 기조의 성소 온기 |
| 4 | Tension in battle | 전투는 강한 명암 대비 + 위험색(레드·앰버)의 독점 사용 — 위험색은 위험에만 쓴다 |
| 5 | Charming forever | 귀여움은 타협하지 않는다. 위협은 크기·자세·발광으로 만든다 |

## 2. 기준 시안 v0.1 해석 — 채택/보류

시안은 최종 UI가 아니라 **화면 흐름·HUD 감각·분위기의 기준**이다 (유키 요청서 §0·§6-10).

|---|---|---|

| Battle: 6슬롯 1×6 + 슬롯명 (Quick Heal~Ring of Light) | **채택** | 초반 6스킬과 1:1 일치 확인 (Lock 07·11 정합) |

| Battle: 보스 옆 "x3" 표기 | 보류 | 쫄 수 표기로 추정 — 전초전 UI에서 재검토 |

## 3. 화면별 분위기 팔레트

| 화면 | 기조 | 감각 |
|---|---|---|
| 타이틀 | 어스름 퍼플 + 사제의 온광 | 위협 속의 온기 |
| 마을·여관 | 크림 · 골드 · 새싹 그린 | 귀환의 안도 |
| 기도소 | 스테인드글라스 광 · 딥블루 | 고요한 결의 |
| 길드 게시판 | 양피지 · 인장 레드 | 모험의 문턱 |
| 전투 | 저채도 배경 + 위험 레드·앰버 고대비 | 긴장 — HUD가 주인공 |
| 결과 (승) | 햇살 골드 | 안도와 자랑 |
| 결과 (패) | 새벽 블루 | 자책이 아닌 분석 |
| 기록실 | 촛불 앰버 · 서고 브라운 | 통찰 |

전투 배경은 **상단 2/3에 정보 밀도를 몰고 하단 1/3(레이드 프레임·슬롯 뒤)은 저밀도**로 — HUD 판독 보호.

루미(나노바나나2) 특성 2가지를 전제로 작성했다: ①투명 PNG 불가 → **마젠타 크로마키가 정석** ②부정문에 약함 → "없어야 할 것"이 아니라 **"있어야 할 상태"를 서술**.

### 7-1. 캐릭터·보스 공통 프리픽스 (매 발주 앞에 붙임)

```
몽글몽글한 2.5등신의 귀여운 동화풍 JRPG 캐릭터.
부드러운 외곽선, 따뜻한 채색, 큰 눈.
배경은 순수한 단색 마젠타(#FF00FF) 하나로 화면 전체를 완전히 균일하게 채운다.
배경은 그라데이션이나 무늬 없이 평평한 한 가지 색으로만 유지한다.
캐릭터 발밑까지 배경색이 그대로 이어진다.
포즈들은 서로 닿지 않게 간격을 넉넉히 띄우고, 화면 가장자리와도 여백을 둔다.
화면에는 캐릭터만 존재하고, 나머지 공간은 전부 배경색으로 남긴다.
의상과 소품은 크림·골드·그린·블루 계열로만 칠한다. (배경 분리용 — 자홍·분홍 계열 배제)
```

### 7-2. 캐릭터 발주 예시 — 사제

```
(공통 프리픽스)
크림색 후드 로브에 금색 자수 테두리, 새싹이 감긴 나무 지팡이,
가슴에 작은 잎사귀 브로치를 단 온화한 미소의 소녀 사제.
포즈 5개를 가로로 나란히:
1) 두 발로 서서 지팡이를 짚은 대기
2) 지팡이를 앞으로 들어 올려 빛을 모으는 시전
3) 두 손을 모으고 눈을 감은 기도
4) 움찔 몸을 웅크린 피격
5) 지팡이를 하늘로 든 활짝 웃는 승리
```

### 7-3. 보스 발주 예시 — 대지뿌리 골렘

```
(공통 프리픽스 — '캐릭터'를 '보스 몬스터'로 바꿔 적용)
이끼와 새싹이 자란 둥근 바위 골렘. 가슴 중앙에 호박색으로 빛나는 코어.
귀엽지만 크고 묵직한 위압감. 짧은 팔다리, 큰 주먹.
포즈 5개를 가로로 나란히:
1) 우두커니 선 대기
2) 두 주먹을 높이 들어 내려찍기 직전
3) 코어가 강하게 빛나는 시전
4) 몸의 균열마다 붉은 빛이 새어 나오는 광폭
5) 무릎 꿇고 주저앉은 격파
```

### 7-4. 배경 발주 (크로마키 불필요 — 풀블리드)

```
모바일 세로 화면용 배경 일러스트, 9:19.5 세로 비율, 화면 가득.
따뜻한 동화풍 JRPG 배경, 부드러운 빛.
[장소 서술 — 예: 새싹빛이 감도는 성소 마을의 광장.
아기자기한 여관과 예배당, 게시판이 보이는 전경.]
화면 하단 1/3은 UI가 올라갈 영역이므로 단순한 지면과 낮은 밀도로 비워둔다.
```
