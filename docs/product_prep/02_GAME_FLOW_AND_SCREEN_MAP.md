# 02 · GAME FLOW & SCREEN MAP — Seed Healer

**Product Prep Pack 01 (2/7)** · 기준: Lock 05·06·09·11 + 기준 시안 v0.1의 화면 흐름 · 작성: 렌 · 2026-07-04

---

## 1. 전체 게임 Flow

기준 시안 v0.1의 흐름(Title → Town → Contract → Setup → Battle → Result → Back to Town)을 골격으로 채택한다.

```
타이틀
 ├─ (첫 실행)   프롤로그 도입 → 프롤로그 전투 → 마을 도착
 └─ (이어하기)  마을 메인

마을 메인 ──┬── 여관 (동료 관리 · 3인 편성)
            ├── 기도소 (스킬 습득 · 6슬롯 장착 · 강화)
            ├── 길드 게시판 (계약 선택 · 위험 태그)
            ├── 기록실 (리포트 · 도감 · 실패 분석)
            └── 시장/공방 (재료 · 소모품 · 선물 · 외형)

길드 게시판 → 계약 준비(최종 확인) → 전투 진입(연출) → 전투 HUD
 → 결과 화면 → (기록실 권유) → 마을 메인
```

**루프 문장:** 마을 → 계약 → 준비 → 전투 → 결과 → 기록실 → 마을.
실패 시에도 동일: 실패 → 리포트 → 마을 복귀 → 재준비 → 재도전 (Lock 09).

## 2. 화면 전환 원칙 (Lock 05: 감성은 풍부하게 · 조작은 짧게 · 선택은 선명하게)

- 마을 내 시설 이동은 **1탭**. 시설 → 마을 복귀도 1탭 (뒤로가기 일관).
- 게시판에서 전투까지 **최단 3탭** (계약 선택 → 출발 확인 → 진입). 준비 화면에서 여관/기도소로 바로가기 제공.
- 결과 → 마을 **1탭**, 결과 → 다음 계약 **1탭** (시안의 [Back to Town]/[Next Contract] 채택).
- 첫 방문 화면에는 1회성 안내 오버레이(탭으로 해제). 강제 잠금은 초반 30분 커리큘럼(04 문서) 구간에만.

## 3. 화면별 명세 (13종)

각 화면은 ①목적 ②감정/분위기 ③주요 UI ④플레이어의 선택 ⑤필요 이미지·에셋 ⑥프로토 P1 최소 요소로 정리한다.

---

### 3-1. 타이틀

- **목적:** 세계의 톤 각인 + 즉시 진입. "따뜻한 세계에 위협이 있다"를 한 장으로.
- **감정/분위기:** 어스름 속의 온기. 시안처럼 보스 실루엣을 배경에, 파티(사제 중심)를 전면에.
- **주요 UI:** 로고, `Tap to Begin`, 하단 소형 아이콘(설정 · 계정 · 도감).
- **플레이어의 선택:** 시작/이어하기.
- **필요 에셋:** 타이틀 배경 1(보스 실루엣 포함), 로고, 파티 키비주얼(배경에 합칠 수 있으면 1장으로).
- **P1 최소:** 로고 텍스트 + 시작 버튼 + 이어하기(저장 존재 시).

### 3-2. 프롤로그 도입

- **목적:** 30초 안에 "나는 사제, 공격하지 않는다, 그런데 내가 필요하다"를 각인.
- **감정/분위기:** 성소의 평화 → 침입의 균열. 짧고 굵게.
- **주요 UI:** 컷 3~5장 + 대사 1~2줄씩, `건너뛰기`.
- **플레이어의 선택:** 진행/스킵.
- **필요 에셋:** 성소/예배당 배경 1, 컷 일러스트(초기에는 배경+캐릭터 초상 조합으로 대체 가능).
- **P1 최소:** 텍스트 카드 3장 + 스킵.

### 3-3. 프롤로그 전투

- **목적:** 빠른 치유 · 보호막 · 정화를 각 1회 이상 **성공 체험** (Lock 11). 상세 설계는 04 문서.
- **감정/분위기:** 긴박하지만 실패 없는 첫 성공. 패배 조건 없음.
- **주요 UI:** 본 전투 HUD의 축소판 — 3슬롯만 활성, 나머지 잠금 표시. 상황별 프롬프트(해당 슬롯 발광 + 한 줄 안내).
- **플레이어의 선택:** 어떤 순간에 어떤 스킬을 누를지 (프롬프트 유도).
- **필요 에셋:** 프롤로그 보스(어스름 이리 우두머리 · 워킹명) + 이리 졸개, 성소 전투 배경.
- **P1 최소:** 본 전투 HUD 재사용 + 슬롯 잠금 + 프롬프트 3종.

### 3-4. 마을 메인 (워킹명: 새싹빛 성소)

- **목적:** 허브. "다음 전투의 답안을 짜는 장소" (Lock 05).
- **감정/분위기:** 따뜻함 · 안전 · 귀환의 안도. 시안의 Sanctuary of Sproutlight 전경 감각.
- **주요 UI:** 마을 전경 위 5시설 진입 버튼(여관 · 기도소 · 게시판 · 기록실 · 시장), 상단 재화 4종, 하단 탭(홈 · 파티 · 계약 · 가방 · 메뉴 — 시안 채택), 다음 행동 추천 말풍선 1개(예: "게시판에 새 계약이 붙었어요").
- **플레이어의 선택:** 어느 시설로 갈 것인가. (추천은 1개만 — 선택은 선명하게)
- **필요 에셋:** 마을 전경 배경 1, 시설 아이콘 5, 재화 아이콘 4.
- **P1 최소:** 시설 버튼 5(시장은 "준비 중") + 재화 표시.

### 3-5. 여관

- **목적:** 동료 관리 · 영입 이벤트 · **3인 파티 편성** (Lock 01·06).
- **감정/분위기:** 벽난로, 동료들의 휴식. 인연이 쌓이는 곳.
- **주요 UI:** 로스터 그리드(보유는 초상, 미보유는 실루엣+획득 힌트), `3/3` 편성 슬롯, 직업 필터 5, 동료 상세 카드(역할 한 줄 · 인연 단계 · 특성), `편성 확정`. 시안의 Inn(Select 3 Allies) 구조 채택.
- **플레이어의 선택:** 15종(초반 5종) 중 3인 조합. 인연 성장 투자(인연 실타래).
- **필요 에셋:** 여관 배경 1, 동료 초상(우선 5종), 직업 아이콘 5.
- **P1 최소:** 보유 5명 중 3인 선택 + 확정 + 역할 한 줄 표시.

### 3-6. 기도소

- **목적:** 사제 스킬 습득 · **6슬롯 장착** · 강화 (Lock 04·07·08).
- **감정/분위기:** 스테인드글라스, 고요한 결의. 사제의 방.
- **주요 UI:** 6슬롯(시안은 원형 배치 — 원형 vs 리스트형은 판단 필요 항목), 보유 스킬 목록(액티브/패시브 탭), 스킬 상세(역할 · 시전 유형 · 강화 단계 MAX3 · 택1 특성), `프리셋 저장`, `자동 장착`(초심자용). 시안의 Chapel(Skill Loadout) 구조 채택.
- **플레이어의 선택:** 20종(초반 7종) 중 6개 장착. 기도 조각 투자처.
- **필요 에셋:** 기도소 배경 1, 스킬 아이콘(우선 7종).
- **P1 최소:** 7스킬 중 6개 장착 + 저장.

### 3-7. 길드 게시판

- **목적:** 보스/스테이지 선택. **위험 태그를 읽고 답안을 구상하는 출발점** (Lock 05·09).
- **감정/분위기:** 모험의 문턱. 양피지 · 도장 · 현상서 감각. 시안의 Contract 보드 채택.
- **주요 UI:** 계약 카드(보스 초상 · 난이도 뱃지 · **위험 태그 2~3개** · 얕은 힌트 한 줄 · 보상 미리보기), `계약`/`현상` 탭, 도감 진척 뱃지(트라이할수록 태그·힌트가 늘어남 — 정답 전체 공개 금지).
- **플레이어의 선택:** 어떤 계약을 받을 것인가. 태그를 보고 편성/장착을 역산.
- **필요 에셋:** 게시판 배경 1, 보스 초상 4(초반), 위험 태그 아이콘 ~9종, 난이도 뱃지 3~4종.
- **P1 최소:** 계약 카드 3장(대지뿌리 골렘 · 서리꽃 마녀 · 메마른 고목) + 선택.

### 3-8. 계약 준비

- **목적:** 출발 전 최종 확인. 위험 태그 ↔ 현재 답안(파티 · 6슬롯)을 한 화면에서 대조.
- **감정/분위기:** 숨 고르기. 문 앞에서 장비를 점검하는 순간.
- **주요 UI:** 위험 태그 재표시, 현재 파티 3인 초상, 현재 6슬롯, `[여관에서 변경]` `[기도소에서 변경]` 바로가기, `출발`.
- **플레이어의 선택:** 이대로 갈 것인가, 답안을 고칠 것인가.
- **필요 에셋:** 없음(기존 요소 조합).
- **P1 최소:** 요약 1화면 + 출발 버튼.

### 3-9. 전투 진입

- **목적:** 전환 연출 + 로딩 은닉 + 마지막 긴장 조성.
- **감정/분위기:** 화면이 어두워지고, 심장 박동. 마을의 온기에서 전장의 냉기로.
- **주요 UI:** 보스 등장 연출(이름 + 위험 태그 1개 강조), 1.5초 내외.
- **플레이어의 선택:** 없음(관람).
- **필요 에셋:** 보스 등장 포즈(전투 시트에서 재활용).
- **P1 최소:** 페이드 + 보스명 표시.

### 3-10. 전투 HUD

- **목적:** 코어 플레이. **상세 스펙 전체는 03 문서.**
- **감정/분위기:** 귀엽지만 위험이 읽힌다. 긴장과 판단.
- **주요 UI:** 보스 HUD / 시전·예고 / 전장 / 레이드 프레임 4칸(1번=사제 본인) / 6스킬 슬롯 / 마나 바.
- **플레이어의 선택:** 대상 지정, 스킬 선택, 시전 취소, (초반 이후) 우선순위 판단 전부.
- **필요 에셋:** 03·06 문서 참조.
- **P1 최소:** 03 문서의 P1 체크리스트.

### 3-11. 결과 화면

- **목적:** 판정 + **거울** + 다음 행동 안내 (Lock 10, 시안 Result 채택).
- **감정/분위기:** 승리 = 햇살과 안도 / 패배 = 차분한 새벽. **자책이 아니라 분석의 톤.**
- **주요 UI:** 승패 배너, 핵심 지표(클리어 타임 · 받은 피해 · 총 회복량), 보상(재화 · 해금), **하이라이트 칩 최대 3개**(예: 전원 생존 / 예방의 손 / 구원의 순간 / 완벽 정화 / 마나 장인 — 스타일이 비치는 칩), 다음 제안 1개(시안의 "Try Frostbloom Witch" 감각), `[마을로]` `[다음 계약]` `[기록실에서 자세히]`.
- **플레이어의 선택:** 다음 행동(마을/재도전/분석).
- **필요 에셋:** 결과 배경(승/패 2종 또는 틴트 차이), 하이라이트 칩 아이콘.
- **P1 최소:** 승패 + 지표 3종 + 보상 + 버튼 2개.

### 3-12. 기록실

- **목적:** 전투 리포트 아카이브 · 보스 도감 · **실패 원인 분석** (Lock 05·09·10: 실패는 분석).
- **감정/분위기:** 서고, 촛불, 사서의 조용한 응원.
- **주요 UI:** 최근 전투 리포트(타임라인 요약 · 위기 순간 톱3 · 스타일 지표: 오버힐% / 예방힐 비중 / 평균 반응 — R&D 리포트 문법 승계), 보스 도감(트라이로 항목이 열리는 카드: 패턴 · 대처 힌트), 실패 시 **원인 한 문장**(예: "52초 강타를 보호막 없이 받았다").
- **플레이어의 선택:** 무엇을 복기할지, 도감의 어느 힌트를 열람할지.
- **필요 에셋:** 기록실 배경 1, 도감 카드 프레임.
- **P1 최소:** 직전 전투 리포트 1장(지표 + 위기 1개 + 원인 문장).

### 3-13. 시장/공방

- **목적:** 강화 재료 · 소모품 · 동료 선물 · 외형/장식 (Lock 05). 소모품이 사제 역할/기믹을 대체하지 않는 선 (Lock 10).
- **감정/분위기:** 활기, 흥정, 마을의 생활감.
- **주요 UI:** 상점 탭 4(재료/소모품/선물/외형), 골드 소비.
- **플레이어의 선택:** 골드의 사용처.
- **필요 에셋:** 시장 배경 1, 아이템 아이콘 세트.
- **P1 최소:** **P1 범위 제외.** 마을에 버튼만 두고 "준비 중" 표시 (07 문서 참조).

## 4. 화면 ↔ 에셋 교차 요약

배경 9종(타이틀 · 성소/예배당 · 마을 · 여관 · 기도소 · 게시판 · 전투 · 결과 · 기록실)이 13개 화면을 전부 커버한다. 프롤로그 도입/전투는 성소 배경 공유, 계약 준비/전투 진입은 기존 요소 조합으로 배경 신규 0. 상세 발주 계획은 06 문서.
