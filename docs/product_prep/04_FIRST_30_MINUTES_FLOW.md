## 2. 전투 상세 (4종)

각 전투는 유키 지정 8항목(목적 / 가르치는 행동 / 사용 가능 동료 / 사용 가능 사제 스킬 / 보스 패턴 / 예상 길이 / 클리어 보상)과 **별도 항목의 손맛 튜닝 포인트**로 정리한다. 모든 패턴은 고정 타임라인 · 조건 기반(난수 0, R&D 승계).

### 2-1. 프롤로그 — 「성소의 밤」

| 항목 | 내용 |
|---|---|
| 전투 목적 | "나는 공격하지 않는데 전투가 돌아간다 — 그리고 내가 없으면 무너진다"를 몸으로 확인. 3스킬 각 1회 이상 **성공 체험** |
| 가르치는 행동 | ① 떨어지는 HP에 **빠른 치유** ② 예고 공격에 **보호막 선대응** ③ 디버프에 **정화** |
| 사용 가능 동료 | 방패 전사 · 차단 도적 · 화염 마법사 + **성소의 노기사**(임시 4번째 — Lock 01 특수전 조항 활용, 렌 제안) |
| 사용 가능 사제 스킬 | 빠른 치유 · 보호막 · 정화 (3슬롯 활성, 나머지 자물쇠 표시) |
| 보스 패턴 | **어스름 이리 우두머리** + 졸개 2. ①물어뜯기: 도적 대상 출혈 도트 → 정화 프롬프트 ②도약 강타: 예고 2s, 전사 대상 → 보호막 프롬프트 ③평타: 잔잔한 유지 피해 → 빠른 치유 프롬프트 |
| 예상 길이 | 전투 60~90초 (도입 컷 포함 **3:00 이하** — Lock 11 하드 기준) |
| 클리어 보상 | 골드 소량 + 서사(마을 도착). **패배 조건 없음** (첫 경험 보호). 전투 후 노기사가 부상으로 이탈 — "이제 자네가 이 아이들을 지켜주게" → 3인 체제의 서사적 동기 부여 |

**손맛 튜닝 포인트 (별도)**
- 프롬프트 방식: 강제 슬로우 vs 일시정지 vs 슬롯 발광만 — 손을 얼마나 멈추는가
- 첫 힐의 짜릿함: 플로팅 숫자 크기 · SFX · 프레임 반응 (§6a 다이얼)
- 3분 초과 여부 / 지루한가·너무 급한가
- 패배 없음이 긴장을 죽이는지 (노기사 HP가 떨어지는 연출로 보완 가능)

### 2-2. 계약 1 — 대지뿌리 골렘 「무너지는 자를 붙잡는 법」

시안 게시판 태그 **[강타 · 광역 · 속박]** (Heavy Smash / Party Damage / Root) 그대로 채택.

| 항목 | 내용 |
|---|---|
| 전투 목적 | 탱커 케어 + 보호막 선대응이 **답이 되는** 첫 진짜 시험 (Lock 11) |
| 가르치는 행동 | 강타 예고 읽기 → 보호막 겹치기 → 강타 직후 낙폭 복구. 속박 정화 복습. **돌진동으로 사제 첫 피격 체험** (Lock 02) |
| 사용 가능 동료 | 방패 전사 · 차단 도적 · 화염 마법사 (보유 3인 = 선택 없음. 편성 화면은 절차 학습으로 통과) |
| 사용 가능 사제 스킬 | 빠른 치유 · 보호막 · 정화 · 구원의 기도 (4슬롯) |
| 보스 패턴 | ①**대지 강타**: 탱커 대상, 예고 1.5s(튜닝 1.2~1.8), 보호막 적중 시 피해 대폭 경감 ②**돌진동**: 전원 소피해 광역 — **사제도 맞는다** ③**뿌리 속박**: 1인 속박+도트 → 정화 해제 ④광폭화 없음 (초반 보호) |
| 예상 길이 | 2:30 기준 (Lock 09 보스전 기준) |
| 클리어 보상 | 골드 + 기도 조각 + **[스킬] 지속 회복**. 직후 여관 이벤트: **치유 토템 주술사** 합류 ("골렘 소동을 듣고 찾아온" — Lock 06 ①스토리 합류 체험) |

**손맛 튜닝 포인트 (별도)**
- 강타 예고 시간 1.2~1.8s — 읽고 반응할 수 있는 최소·최대
- 노보호막 강타의 아픔: 사망은 아니되 공포는 남게
- 돌진동 빈도·강도: 사제 피격의 첫인상 (겁주기 vs 짜증)
- 2:30 체감 길이 / 마나 압박은 첫 계약이므로 **여유 있게** (압박은 계약 3의 몫)

### 2-3. 계약 2 — 서리꽃 마녀 「정화가 차단을 만든다」

시안 게시판 태그 **[차단 · 빙결 · 저주]** (Interrupt / Freeze / Curse) 그대로 채택. R&D EP05 J3 연계의 직계 승격.

| 항목 | 내용 |
|---|---|
| 전투 목적 | 씨앗의 심장 이식 — **아군의 실수·위기를 사제가 되돌리는 개입 창** 학습 (Lock 11 "정화가 차단을 만든다") |
| 가르치는 행동 | 차단 가능/불가 시전바 구분 · 낙인→정화→차단 연계 · HoT 선배포 |
| 사용 가능 동료 | **첫 자유 편성** (보유 4인 중 3인 = 4조합). 게시판 태그 「차단」이 도적 편성을 안내 — "태그가 답안을 이끈다" 체험. 도적 미편성 시 클리어 가능하되 크게 어려움 (의도된 밸런스) |
| 사용 가능 사제 스킬 | 5종 보유 (시작 4 + 지속 회복) — 슬롯 여유, 전부 장착 |
| 보스 패턴 | ①**서리 화살**: 평타 ②**얼음 보라**: 차단 가능 시전 2.5s → 도적 차단 ③**겨울의 낙인**: 도적 대상 빙결 낙인 5s — **정화하면 도적이 풀려나 차단 가능** (J3 직계) ④**만개하는 서리**: 차단 불가 시전 → 전원 피해 — HoT 선배포·회복 로테이션으로 받아내기 |
| 예상 길이 | 2:30 |
| 클리어 보상 | **[스킬] 빛의 고리** + **[시설] 기록실 개방**. 첫 리포트 열람 유도: "낙인을 N초 만에 정화해 차단 M회를 만들었다" (리포트=거울) |

**손맛 튜닝 포인트 (별도)**
- 낙인 지속 4~6s = 연계 창의 넓이 — 성공했을 때 "내가 해냈다"가 남는 폭
- 낙인 방치 벌점의 체감 (차단 실패의 아픔이 다음 트라이를 부르는가)
- 차단 불가 국면의 회복 압박 — 빛의 고리가 **없어서 답답한가** (해금 동기 설계)
- 도적 미편성 난이도 / 전투 호흡이 계약 1과 다르게 느껴지는가

### 2-4. 계약 3 — 메마른 고목 「마나는 유한하다」 (전초전 포함)

| 항목 | 내용 |
|---|---|
| 전투 목적 | **마나 경제가 이 게임의 숨은 난이도**임을 각인. 전초전 이월 구조 첫 체험 (Lock 09) + 구출 획득 체험 (Lock 06) |
| 가르치는 행동 | 오버힐 절제 · 비시전 회복 창 만들기 · 전초전에서 아끼기 |
| 구조 | 전초전 「**시든 새싹 무리**」(45초 일반전) → [구출] 덩굴 우리의 **덫 사냥꾼** → 편성 확인(즉시 투입 가능) → 본전 「**메마른 고목**」(2:30). **마나/쿨다운 이월** |
| 사용 가능 동료 | 자유 편성 (보유 4 → 구출 후 5) |
| 사용 가능 사제 스킬 | 6종 전부 — **첫 만슬롯 경험** |
| 보스 패턴 (본전) | ①**마른 할큄**: 평타(약) ②**갈증의 안개**: 장시간 전원 소량 지속 피해 — **마나 배수구** ③**뿌리 뽑기**: 강타 복습 ④**시듦의 씨앗**: 폭탄형 3s 퓨즈 → 정화 최우선 |
| 예상 길이 | 45s + 2:30 (총 ~3:15 — 특수 장기전 감각의 입구) |
| 클리어 보상 | **[패시브] 깊은 호흡** (마나 시험의 서사적 정답 — 배치는 렌 제안) + **덫 사냥꾼 합류** (5직업 완성) + **보스 증표 첫 획득** |

**손맛 튜닝 포인트 (별도)**
- 전초전 적정 소모율: 전량 소진 시 본전에서 벌칙이 체감되는가
- **이월 규칙**: 마나 전량 이월 vs 막간 일부 회복 (판단 필요 항목 — 07 문서)
- 지속 피해 강도: 지루함과 질식 사이
- 오버힐 경고 UI 필요 여부 / 총 3:15가 긴가

## 6. 손맛 튜닝 후보 총괄 (유키 6질문 프레임)

| 질문 | 주요 다이얼 |
|---|---|
| 지루한가? | 프롤로그 프롬프트 밀도 / 계약 3 지속 피해 구간의 단조로움 |
| 너무 급한가? | 강타 예고 1.2~1.8s / 낙인 연계 창 4~6s / 폭탄 퓨즈 3s |
| 전투 호흡이 좋은가? | 2:30 기준 길이 / 패턴 간격 / 전초전→본전 막간 길이 |
| 힐 버튼이 짜릿한가? | 플로팅·SFX·프레임 반응 강도 (§6a 손맛 툴킷 다이얼) |
| 마나 압박이 적당한가? | 계약 1(여유)→3(압박) 경사 / 이월 규칙 / 자연 회복량 |
| 해금 속도가 빠르거나 느린가? | 계약당 1스킬+1동료 리듬 / 깊은 호흡 배치 시점 |

전부 **구조는 유지, 값만 조정** 가능한 항목으로 설계했다 (Lock 07: "스킬 구조는 설계로 잠그고, 스킬의 호흡은 손맛으로 맞춘다").
