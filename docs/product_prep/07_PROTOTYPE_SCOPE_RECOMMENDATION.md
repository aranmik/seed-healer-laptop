## 1. P1의 목표 문장

**"이미지 없이도 손맛이 성립하는가"를 먼저 증명하고, 이미지는 교체로 붙인다.**
루미 발주(06 문서 Set 01·02)와 완전 병렬 — 프로토는 플레이스홀더(도형·이모지)로 시작하고, 에셋은 manifest 교체로 결합한다.

## 2. P1 범위 (In / Out)

### In — 유키 권장 9화면 + α

| # | 화면 | 최소 구현 수준 |
|---|---|---|
| 1 | 타이틀 | 로고 텍스트 + 시작/이어하기 |
| 2 | 마을 메인 | 시설 버튼 5 (시장은 "준비 중") + 재화 표시 |
| 3 | 길드 게시판 | 계약 카드 3장 (골렘·마녀·고목) — 태그·힌트·보상 |
| 4 | 여관 3인 편성 | 보유 5명 중 3인 선택 + 역할 한 줄 |
| 5 | 기도소 6슬롯 | 보유 7스킬 중 6개 장착 + 저장 |
| 6 | 첫 계약 강타 보스 | 대지뿌리 골렘전 완주 (04 문서 2-2 스펙) |
| 7 | 전투 HUD | 03 문서 P1 체크리스트 전체 |
| 8 | 결과 화면 | 승패 + 지표 3종 + 보상 + 하이라이트 칩 |
| 9 | 기록실 | 직전 전투 리포트 1장 (지표·위기 1개·원인 문장) |

- **콘텐츠**: 동료 5종(초반 라인업) · 사제 스킬 7종(액티브 6+깊은 호흡) · 보스는 골렘 1종 완성 기준.
- **저장**: localStorage (try/catch 필수 · 스키마 버전 필드 포함).
- **검증 편의**: P1은 해금 시퀀스 없이 동료 5·스킬 7 **전량 지급 상태**로 시작 — 여관의 "선택"(5C3)과 기도소의 "선택"(7중6)이 성립해야 판단 부담을 검증할 수 있다. 서사 해금은 P1.5의 몫.

### P1.5 — 초반 30분 커리큘럼 완주 (P1 통과 후)

프롤로그 전투 + 서리꽃 마녀 + 메마른 고목(전초전 포함) + 해금 시퀀스(04 문서 타임라인) + 위기 힌트 시스템.

### Out — P1·P1.5 공통 제외

시장/공방(버튼만) · 동료 잔여 10종 · 스킬 잔여 13종 · 성장 강화/특성 택1(획득만, 강화는 스텁) · 챕터 구조 · 현상 탭 · 프리셋 다중 저장.

## 3. 구현 순서 — 5 Phase

| Phase | 내용 | 완료 판정 |
|---|---|---|
| **A. 전투 코어 개조** | R&D 코어 이식 후 ①아군 4→3 ②사제를 레이드 프레임 1번 칸으로 이동 + 피격 경로 도입(광역·어그로 이탈·후열 찍기) ③전투 길이 75s→150s **마나 곡선 재설계** ④골렘 타임라인 작성 | 봇 하니스 4형제 통과 + 골렘전 손판정 1회 |
| **B. 결과/리포트** | 지표 3종 + 하이라이트 칩 + 원인 문장 → 기록실 최소 리포트 | 패배 후 원인 문장이 정확히 뜸 |
| **C. 마을 4화면** | 게시판(카드 3) → 계약 준비 → 여관 3/3 → 기도소 6슬롯 | 게시판→전투 최단 3탭 |
| **D. 루프 + 저장** | 타이틀 이어하기 · 진행 저장 · 재도전 루프 | 앱 재시작 후 이어하기 성공 |
| **E. 이미지 교체** | Set 01 수령 시 전투 화면부터 manifest 스왑 | 플레이스홀더/이미지 토글 가능 |

Phase A~D는 이미지 0장으로 완결 — **루미 발주와 병렬성 100%.**

## 5. 이미지 결합 순서 (Set 도착 시)

## 8. 나라님/유키PD 판단 필요 항목

1. **P1 파일 체제**: 단일 HTML 시작 → 이미지 시 repo 전환 (렌 권고) vs 처음부터 repo
2. **전투 대상 지정**: 프레임 선탭 → 스킬 (렌 권고 · R&D 검증) vs 스마트 캐스트
3. **기도소 6슬롯 UI**: 원형(시안) vs 리스트형
4. **자원 명칭**: '마나' 유지 (렌 권고 — 성장축 '사제 신앙'과 용어 충돌 회피) vs '신앙(Faith)'
5. **한글 워킹명 컨펌**: 새싹빛 성소 / 어스름 이리 / 대지뿌리 골렘 / 서리꽃 마녀 / 메마른 고목
6. **프롤로그 4번째 동료** — 성소의 노기사(부상 이탈 서사) 채택 여부
7. **게시판 갱신 타이머** 채택 여부 (§7 리스크 참조)

**P1의 끝 그림:** 폰 세로 화면에서 — 게시판에서 골렘 계약을 받고, 여관에서 셋을 고르고, 기도소에서 여섯을 걸고, 2분 30초를 손끝으로 버텨내고, 결과 화면의 거울을 보고, 다시 마을로 돌아오는 **한 바퀴가 저장과 함께 돈다.** 그 바퀴가 재미있으면, 이미지를 입힌다.
