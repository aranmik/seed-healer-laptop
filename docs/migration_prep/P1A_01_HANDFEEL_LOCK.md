# P1A · 01 · HANDFEEL LOCK — Seed Healer

**Migration Prep Pack 01 (1/11)** · 작성: 렌 · 2026-07-04
이 문서는 **현재 재미를 만든 기준을 잠근다.** 노트북 재구현 시 여기 적힌 감각이 사라지면 그 재구현은 실패다.

> **Seed Healer의 재미는 완벽한 정답 입력이 아니라, 죽기 직전까지 붙잡는 힐러 손맛이다.**

---

## 1. 핵심 문장 (변경 불가)

```
나는 공격하지 않는다.
하지만 전투의 결과는 내 손에 달려 있다.

죽은 뒤에 복구하는 게임이 아니라,
죽기 직전까지 붙잡는 게임.
```

---

## 2. 나라님 첫 재미 판정 요약 (근거 데이터)

| 항목 | 판정 |
|---|---|
| 종합 | "진짜 재미있다" — FUN PASS |
| 난이도 첫인상 | "딱 좋다 — 전사 죽어서 쫄깃했고 다음엔 살리고 싶다" (상위 0.1% 레이더 보정 포함) |
| 대표 클리어 | **강타 대응 0/6**, 방패 전사 1명 사망, 남은 둘로 골렘 HP 밀어 승리 (2:26) |
| 가장 신경 쓰인 것 | ①정화 타이밍 ②빛의 고리(전원 힐)가 안 나가서 "캔슬됐다"고 오인지 — 실제로는 쿨다운/마나. **첫트라 인지 문제, 다음엔 쉬울 것**으로 자평 |
| 다음 목표 | "강타에 보호막 제대로 걸어서 전원 생존 클리어" |

**해석**: 재미의 정체는 **"비벼서 이기는 것"**이다. 강타를 다 맞고도(0/6) 이긴 판이 재미있었다는 것은, 이 게임이 **정답 암기 게임이 아니라 실시간 수습 게임**이라는 증거다.

## 3. 잠금 기준값 (LOCK)

| 손맛 | 현재 기준 | 잠금 의미 |
|---|---|---|
| 전투 목표 길이 | **약 150초** (표준 파티 승리 2:25 실측, 허용 120~170s) | 이보다 짧으면 판단 창이 죽고, 길면 늘어진다 |
| 마나 압박 | 표준 파티 좋은 플레이 시 **잔여 15~20%** (스마트 봇 17/100) | "거의 다 썼지만 살짝 남는" 선. 넉넉하면 긴장이 죽는다 |
| 오버힐 | 좋은 플레이 시 **2~7%** (스마트 2%, 기본기 3%) | 오버힐이 낮아야 마나가 남는 인과. 스팸 시 39%로 치솟음 |
| 전사 사망/생존 | 방치 시 확실히 죽되(**노보호막 방치=사망**), 소프트 압박이라 즉사는 아님 | 대가는 오지만 벼랑에서 붙잡을 수 있다 |
| 강타 보호막 | 보호막 적중 시 피해 **약 1/3.5로 경감**(무보호 504 vs 보호 144) | 강력한 이득. 하지만 강제 아님(4항 참조) |
| 생힐로 비비기 | **강타 0/6으로도 승리 가능** | 이 게임의 정체성. 절대 삭제 금지 |
| 마지막 압박 | t≥120s 또는 HP≤25%에서 평타·돌진동 가속(피해 ×1.12), **즉사 없음** | 소프트 인레이지 — 힐 체크로 번역된 DPS 체크 |

## 4. 강타 보호막의 역할 — "이득이되 강요는 아니다" (재미의 핵심 축)

이 게임에서 가장 조심해야 할 지점. 세 문장으로 잠근다.

1. **왜 보호막으로 선대응해야 하는가**
   강타는 예고(1.5s) 후 탱커에게 큰 피해(720). 보호막(360 흡수)을 미리 걸면 피해가 크게 줄어(실측 504→144) 그 뒤 회복 부담이 급감한다. **읽고 준비하면 편해진다** — 이게 정석 보상.

2. **그러나 왜 보호막이 강제 정답이면 안 되는가**
   보호막을 못 걸면 죽는 구조라면, 이 게임은 "예고 보고 버튼 누르기"로 축소된다. 그러면 죽기 직전까지 붙잡는 손맛이 사라진다. 강타는 **아프지만 즉사가 아니다**(탱커 감쇄 30% + HP 1400 + 소프트 압박).

3. **미대응 강타도 생힐로 비빌 수 있어야 하는 이유**
   나라님의 0/6 승리가 살아있는 증거다. 강타를 놓쳐도 빠른 치유·지속 회복·구원의 기도로 **수습할 창**이 남아야 한다. 실패가 곧 사망이 아니라, 실패가 곧 "더 바빠짐"이어야 한다.

> 재구현 검증: 노탱커나 미숙 플레이로도 **가끔은 비벼서 이겨져야** 한다. 항상 이기면 너무 쉬운 것이고, 강타 놓치면 반드시 지면 강요가 된 것이다. **QA에서 봇으로 이 경계를 확인**한다(10 문서).

## 5. 절대 건드리면 안 되는 재미 요소 (동결 목록)

- **결정론**: 난수 0. 같은 입력=같은 결과. 리트라이=학습.
- **사제 피격**: 돌진동은 사제(YOU)도 맞는다. 사제 사망 = 즉시 패배.
- **시전 완료 시 마나 소비**: 시전 취소·무산은 시간 손실만(이중 페널티 금지).
- **부활/사망 방지 없음**: 죽으면 회색+🪦, 되돌리는 UI 없음.
- **구원의 기도 = 사제 전용**: 잃은 HP×0.75+100. 파티 회복 아님, 사망 방지 아님.
- **거부 사유 가독**: 스킬이 안 나가면 이유가 즉시 보인다(슬롯 위 배지). 1차 손판정 최우선 개선분 — 후퇴 금지.
- **소프트 압박**: 마지막 국면은 즉사가 아니라 압박 급상승.

## 6. 재미 회귀 체크 (재구현 후 이 감각이 살아있는가)

- [ ] 첫 판을 아무 생각 없이 해도 **가끔 이기고 가끔 진다** (강요 아님)
- [ ] 강타에 보호막 걸면 **눈에 띄게 편해진다** (이득 성립)
- [ ] 스킬 스팸하면 **후반에 마나로 무너진다** (경제 벽)
- [ ] 전사를 방치하면 **죽지만, 벼랑에서 살릴 수도 있다** (붙잡는 손맛)
- [ ] 리포트가 **내 플레이를 정확히 비춘다** (거울)
- [ ] "다음엔 전원 생존" 같은 **재도전 동기**가 남는다
