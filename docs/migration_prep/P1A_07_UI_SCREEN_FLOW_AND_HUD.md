## 3. 전투 HUD 해부 (세로 6존)

```
┌─────────────────────────────┐
│ A 보스 HUD    이름·HP바·타이머·⏸ │ ~72px
│ B 시전/예고   전조 바 or "다음:__"   │ ~46px
│ C 전장        보스 스프라이트·AoE링·토큰·힌트 │ flex
│ D 레이드 프레임 4칸 (1번=YOU)         │ ~78px×
│ E 스킬 6슬롯  + 대상 표시줄            │
│ F 마나·시전바·로그                    │
└─────────────────────────────┘
```

### A · 보스 HUD
- 난이도 뱃지 + 이름 + 타이머(mm:ss) + 일시정지 버튼.
- HP바: fill + 25%/50%/75% 눈금 + 퍼센트 텍스트.
- (원본) 압박 시 화면 가장자리 붉은 펄스는 전투 루트 컨테이너에 클래스.

### B · 시전/예고 (읽힘의 핵심 1)
- 진행 중 예고가 있으면 **컬러 예고 바**(fill 왼→오, 남은 초, 라벨):
  - 강타: 앰버 바 "🔨 강타! {대상}에게 보호막을" → **보호막 걸리면 녹색 "🛡 강타 대비 완료"**
  - 돌진동: 적색 바 "⚠️ 돌진동 — 전원 피해!(사제 포함)"
  - 속박: 보라 바 "⛓️ 뿌리가 솟는다 → {대상}"
- 없으면 idle: "다음: 🔨 강타 (Ns)" (가장 가까운 패턴) + 압박 시 "마지막 압박!".

### D · 레이드 프레임 — 시그니처 (읽힘의 핵심 2)
**4칸 고정, 1번 칸 = YOU(사제, 금테).** 프레임 1칸 구성:
```
[초상] 이름 [YOU]     ⚔️ 🎯   ← ⚔️ 어그로, 🎯 강타 예고 대상
[HP바: 회복 겹층 + 보호막 겹층(하늘색) + 수치]
[마나 미니바 — 사제만]
[칩: ⛓️속박(확대·맥박) · 🛡보호막잔량 · 🌿HoT]
```
- **탭 = 대상 선택**(선택 시 발광 테두리). 입력 방식: 프레임 선탭 → 스킬 탭.
- 위험 펄스: HP<30% 테두리 맥박.
- **강타 예고 대상**: 프레임 전체가 앰버로 강조+맥박(1차 손판정 개선). 🎯 마크.
- 플로팅 숫자: 피해=적, 회복=녹, 흡수=하늘 (프레임 로컬, 최대 3개).
- 사망: 회색+🪦, 토큰도 쓰러짐. 부활 UI 없음.

### E · 스킬 6슬롯 (읽힘의 핵심 3)
- 슬롯: 아이콘 + 이름 + 마나비용(또는 패시브 "P").
- 상태 표현: GCD 스윕(하단 바) / 쿨다운(어둡게+남은 초 큰 숫자) / 마나 부족(회색+파란 테두리) / 시전 중(테두리 링) / 패시브(점선·반투명).
- **거부 사유 배지**(1차 손판정 최우선 개선): 못 나가면 슬롯 위 **빨간 배지**("6s"/"마나 부족"/"이미 보호막"/"해제할 것 없음"/"대상 없음") + 슬롯 흔들림 + 경고음. GCD는 배지 없이 슬롯 플래시만(소프트).
- 대상 표시줄: 슬롯 위 "대상: {이름} (YOU)".

### F · 자원·시전·로그
- 마나 바: 수치 병기(72/100) + 음소거 버튼.
- **사제 시전바**: 시전 중 표시, **탭=취소**(마나 미소비, 시간만 손실).
- 로그: 최근 3줄(색 구분), 탭하면 확장(12줄).

### 진입/일시정지/종료
- 진입 오버레이 1.3s(보스명·태그).
- 일시정지: 계속/소리토글/포기(→패배 리포트 "전투를 중단했습니다", 보상 없음).
- 종료 오버레이 1.4s(🏆/🌙) → 결과 화면.

## 4. 손맛 툴킷 (연출·SFX — 전부 코드, 이미지/오디오 파일 0)
| 사건 | 화면 | 합성 SFX(WebAudio) |
|---|---|---|
| 평타 | 소흔들림 | 저음 타격 |
| 강타 적중 | 대흔들림+비네트 | 하강 스퀘어 |
| 보호막 흡수 | 하늘색 플래시 | 상승 트라이앵글 |
| 정화 | 녹색 플래시 | 상승 사인 2음 |
| 구원 | 황금 플래시 | 상승 사인 |
| 돌진동 | AoE 링+대흔들림 | 강타류 |
| 사망 | 무채색 플래시 | 하강 톱니 |
| 압박 진입 | 가장자리 붉은 펄스 | 저음 롱 |
| 승/패 | 오버레이 | 상승 아르페지오 / 하강 |

재구현 주의: SFX는 오실레이터 합성(주파수·타입·게인·슬라이드)으로. 음소거 상태면 무음. 첫 사용자 입력에서 AudioContext resume.

## 5. 재구현 최소 순서(UI)
assets 헬퍼 → 전투 HUD(입력·프레임·슬롯·예고 바·거부 배지) → 마을/게시판/여관/기도소 → 결과/기록실 → 진입·일시정지 오버레이. HUD가 먼저여야 코어 검증을 눈으로 할 수 있다.
