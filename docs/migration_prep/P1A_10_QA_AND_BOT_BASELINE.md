## 3. 수동 QA 체크 (나라님/외부 손판정 — 읽힘 & 손맛)

**읽힘 (1차 손판정 개선분 회귀 — 후퇴 금지):**
- [ ] **강타 대상이 보이는가** — 🎯 + 대상 프레임 앰버 강조 + 예고 바 "보호막을"
- [ ] **보호막 대비 완료가 읽히는가** — 보호막 걸면 예고 바가 녹색 "🛡 강타 대비 완료"
- [ ] **스킬 거부 사유가 이해되는가** — 안 나가면 슬롯 위 빨간 배지("6s"/"마나 부족" 등)+흔들림+경고음 (나라님 "캔슬" 오인지 해소분)
- [ ] **속박/정화 타이밍이 보이는가** — ⛓️ 칩 확대·맥박

**손맛 (01 문서 락 회귀):**
- [ ] **마나 압박이 살아 있는가** — 스팸하면 후반 붕괴, 좋은 플레이는 잔여 15~20%
- [ ] **전사 사망 긴장이 살아 있는가** — 방치의 대가는 오되 소프트(즉사 아님)
- [ ] **생힐로 비비는 재미가 남아 있는가** — 강타 놓쳐도 수습해 이길 창이 있음(0/6 승리 재현 가능)

**판정자 보정**: 나라님 = 상위 0.1% 레이더 → 체감보다 한 단계 하향해서 읽을 것. 가능하면 외부 손판정 1인 병행(사제 피격 도입으로 변수 큼).

## 4. 재구현 판정 흐름

```
1) tuning.js + battle.js 이식 → node로 봇 4종 실행
2) §1 표와 대조:
   - P0 패배(~39s) / P1 고갈(~14s)·패배 / P1.5 빠듯 승리(~145s) / P2 승리(마나 10~25)
   - 변형: 깊은호흡 여유·노탱커 패배·저DPS 신승
   - 하나라도 방향이 어긋나면 → tuning/규칙 대조(06·03 문서)
3) coreChecks 9종 통과
4) HUD 붙인 뒤 §3 수동 QA
5) 전부 통과 시 "같은 손맛" — 그다음에만 튜닝 실험 허용
```
