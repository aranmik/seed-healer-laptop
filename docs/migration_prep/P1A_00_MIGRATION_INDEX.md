## 2. 현재 P1-A 상태 요약 (FUN PASS)

- 나라님 실기 판정: **"진짜 재미있다"** — 재미 방향 PASS 확정.
- 대표 사건: 강타 대응 **0/6**(보호막 한 번도 안 걸고)으로 **승리**. 방패 전사 1명 사망, '아슬아슬한 승리'+'위기의 손' 칩 획득.
  - 이 한 판이 증명한 것: ①마지막 압박이 소프트라 강타 다 맞아도 즉사하지 않는다 ②그럼에도 방치하면 전사는 죽는다(대가는 확실) ③**정답 입력이 아니라 손으로 비벼서 이기는 힐러 손맛**이 성립한다.
- 1차 손판정 반영(가독성만, 밸런스 불변): 스킬 거부 사유 배지 승격 / 강타 대상 프레임 앰버 강조 + 예고 바 행동 지시 + 보호막 시 녹색 전환 / 속박 칩 확대.

## 4. 절대 보존해야 할 손맛 (요약 — 상세는 01 문서)

1. **죽기 직전까지 붙잡는 감각** — 완벽 입력 강요 금지. 미스해도 만회할 창이 있어야 한다.
2. **강타 보호막은 "이득"이지 "정답 강요"가 아니다** — 보호막 없이도 생힐로 비빌 수 있어야 한다(0/6 승리가 증명).
3. **마나는 유한하다** — 스팸하면 후반에 무너진다. 아끼는 것도 실력.
4. **전사 사망 긴장** — 방치의 대가는 확실히 오되, 즉사(소프트 압박)는 아니다.
5. **사제도 전장 안에 있다** — 돌진동에 사제도 맞는다.
6. **결정론** — 같은 입력은 같은 결과. 리트라이가 학습이 된다.
7. **거부 사유가 읽힌다** — 왜 스킬이 안 나가는지 즉시 보인다(1차 손판정에서 최우선 개선된 부분).
