> 게시판 태그: **[강타 · 광역 · 속박]** · 힌트 "방패가 버티면 모두가 버틴다." · 목표 길이 약 2:30(표준 파티 실측 2:25).

## 1. 타임라인 개요 (표준 파티 기준 첫 사이클)

```
t=0     전투 시작 (진입 오버레이 1.3s 후)
t=3.0   첫 평타
t=15    첫 강타 (예고 t=13.5부터)
t=20    첫 속박 (예고 t=19부터)
t=25    첫 돌진동 (예고 t=22.5부터)
... 이후 각 패턴이 고유 주기로 반복 ...
t≥120 또는 HP≤25%  → 마지막 압박(가속) 래치
보스 HP 0  → 승리 (표준 ~145s)
```

주기: 평타 5.0s · 강타 24s · 돌진동 28s · 속박 32s (압박 후 평타 3.8s·돌진동 18s로 가속).

## 2. 패턴별 해부

### 2-1. 평타 (auto)
| 항목 | 내용 |
|---|---|
| 발동 조건 | `t >= nextAuto`, 이후 `nextAuto += autoInt(5.0→압박 3.8)` |
| 예고 | **없음** (리듬 유지용 상시 피해 — 예고는 4대 위험에만) |
| 대상 | 현재 어그로(=탱커 우선, 없으면 최고 DPS) |
| 피해/효과 | 300 × 압박배수(1 또는 1.12), 탱커면 30% 감쇄 |
| 대응 스킬 | 빠른 치유·지속 회복으로 탱커 유지 |
| 성공 피드백 | (일상 피해라 별도 강조 없음) 타격 SFX + 소흔들림 |
| 실패 피드백 | 누적되면 탱커 HP 하락 → 위험 펄스 |
| 리포트 기록 | dmgTaken에 합산 |
| 튜닝 위험도 | **MID** — 상시 회복 부담의 바닥. 올리면 힐 여유가 급감 |

### 2-2. 대지 강타 (smash) ★ 핵심
| 항목 | 내용 |
|---|---|
| 발동 조건 | `t >= nextSmash`, 이후 `+= smashInt(24)` |
| 예고 | `smashWind(1.5)s` 전 `tele.smash` 생성. **대상 프레임 🎯 + 앰버 강조**, 상단 예고 바 "🔨 강타! {대상}에게 보호막을", 보스 windup 포즈 |
| 대상 | 현재 어그로(탱커 우선) |
| 피해/효과 | **720** × 압박배수, 탱커 30% 감쇄. 보호막 있으면 흡수(360)만큼 차감 |
| 대응 스킬 | **보호막**(예고 중 대상에 선시전) → 실측 504→144 경감. 또는 직후 빠른 치유/구원으로 복구 |
| 성공 피드백 | 보호막 적중 시 예고 바 **녹색 "🛡 강타 대비 완료"** + 하늘색 플래시 + absorb 플로팅. 리포트 smashShielded++ |
| 실패 피드백 | 큰 흔들림 + 비네트 + 붉은 큰 데미지 플로팅 + 강타 SFX |
| 리포트 기록 | smashTotal, smashShielded (대응률로 표시), 사망 시 원인 문장 |
| 튜닝 위험도 | **HIGH** — 이 게임 난이도의 중심축. 4절 참조 |

### 2-3. 돌진동 (tremor)
| 항목 | 내용 |
|---|---|
| 발동 조건 | `t >= nextTremor`, 이후 `+= tremorInt(28→압박 18)` |
| 예고 | `tremorWind(2.5)s` 전 `tele.tremor` 생성. **바닥 붉은 링 확장** + 상단 바 "⚠️ 돌진동 — 전원 피해!(사제 포함)" |
| 대상 | **전원(아군 3 + 사제 YOU)** |
| 피해/효과 | 130 × 압박배수, 각 유닛에 개별 적용(탱커는 감쇄) |
| 대응 스킬 | 빛의 고리(직후 전체 안정화)·HoT 선배포·자기 대상 케어 |
| 성공 피드백 | (사제 피격 시) YOU 프레임 흔들림 + 화면 비네트 |
| 실패 피드백 | 방치 시 물몸(마법사)·사제 HP 위험 |
| 리포트 기록 | dmgTaken·priestTaken에 합산 |
| 튜닝 위험도 | **HIGH** — "사제도 전장 안에 있다"의 유일한 구현 경로(P1). 간격이 사제 피격 빈도, 피해가 강도 |

### 2-4. 뿌리 속박 (root)
| 항목 | 내용 |
|---|---|
| 발동 조건 | `t >= nextRoot`, 이후 `+= rootInt(32)` |
| 예고 | `rootWind(1.0)s` 전 `tele.root` 생성. 상단 바 "⛓️ 뿌리가 솟는다 → {대상}" |
| 대상 | **비탱커 DPS순 로테이션**(rootRota, 결정론) |
| 피해/효과 | 지속 도트 `rootDps(36)/s` × `rootDur(8)s`. **정화로 즉시 해제** |
| 대응 스킬 | **정화**(대상 프레임 탭 → 정화) — 방치 시간이 곧 추가 피해 |
| 성공 피드백 | 정화 SFX + 녹색 플래시 + 칩 제거. 리포트 rootCleansed++, 해제까지 걸린 시간 누산 |
| 실패 피드백 | ⛓️ 칩(확대·맥박) 유지 + 지속 피해. 만료 시 rootExpired++ |
| 리포트 기록 | rootApplied/rootCleansed/rootExpired, 평균 정화 시간 |
| 튜닝 위험도 | **MID** — 정화 학습 장치. dur·dps가 정화 압박 |

### 2-5. 마지막 압박 (push · 소프트 인레이지)
| 항목 | 내용 |
|---|---|
| 발동 조건 | `t >= pushTime(120)` **또는** `boss.hp/max <= pushHpPct(0.25)` (먼저 오는 쪽, 1회 래치) |
| 예고 | 상단 "마지막 압박!" + **화면 가장자리 붉은 펄스** + 보스 균열 붉은빛 |
| 대상 | 전역(패턴 가속) |
| 피해/효과 | 평타 간격 5.0→3.8s, 돌진동 28→18s, 모든 피해 ×`pushDmgMul(1.12)` |
| 대응 스킬 | 남은 마나 계산 후 총력 유지 |
| 성공 피드백 | 버텨서 보스 HP 0 → 승리 |
| 실패 피드백 | 가속된 피해에 붕괴 |
| 리포트 기록 | (별도 플래그 없음, 사망 시각·원인에 반영) |
| 튜닝 위험도 | **HIGH** — **절대 즉사 금지**(소프트 유지). pushDmgMul을 키우면 하드 인레이지가 되어 손맛 파괴 |

## 4. 강타 설계 3원칙 (재미의 심장 — 01 문서 §4 재확인)

1. **왜 보호막으로 선대응해야 하는가**: 예고 1.5s + 큰 피해 720. 보호막(360)을 미리 걸면 504→144로 경감 → 회복 부담 급감. 읽고 준비하면 편해진다.
2. **왜 강제 정답이면 안 되는가**: 못 걸면 죽는 구조 = "예고 보고 버튼 누르기"로 축소. 강타는 아프되 즉사 아님(탱커 감쇄+HP 1400+소프트 압박).
3. **왜 미대응도 비벼져야 하는가**: 나라님 **0/6 승리**가 증거. 강타 놓쳐도 빠른치유·HoT·구원으로 수습 창이 남는다. 실패=사망이 아니라 실패=더 바빠짐.

**튜닝 시 지켜야 할 경계**: smashDmg를 올려 "노보호막=즉사"가 되면 강요, 내려서 "맞아도 티 안 남"이 되면 무의미. 현재 720이 "아프지만 비벼지는" 선. 봇으로 노탱커=패배 / 표준 스마트=승리 경계 확인(10 문서).
